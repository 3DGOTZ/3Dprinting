Que tipo de impresora tengo?

BOWDEN O EXTRUSION DIRECTA

En el mundo de las impresoras 3D especificamente en FDM podremos encontrar dos diferentes tipos de impresoras 3D, si bien, ambas cumplen el mismo objetivo y sistema de impresion, tienen una caracteristica diferente que influye en su proceso de ejecucion y la elecion de materiales. Los cuales son Sistema Bowden y sistema de extrusion directa.
SISTEMA BOWDEN; Si una impresora menciona que es de extrusion mediante sistema bowden, significa que para poder extruir el filamento ingresado necesita realizar un recorrido por un tubo de teflon, este tipo de sistema se encuentra en impresoras como:

  1. Creality Ender 3 (modelos básicos como la Ender 3, Ender 3 Pro, y Ender 3 V2).
  2. Anycubic i3 Mega S.
  3. Artillery Sidewinder X1 y X2.
  4. Creality CR-10 y sus variantes (CR-10S, CR-10 V3, etc.).
  5. Anycubic Chiron.
  6. Tronxy X5SA.
El material mas recomendado para utilizar en este tipo de impresoras, es el PLA ya que debido a sus caracteristiacas no supone un gran problema para este sistema. Sin embargo, si queremos hacer uso de otros tipos de materiales como el ABS o TPU nos encontraremos en un dilema debido a su principal problema; las retracciones En el uso de TPU asi como ABS necesitan de mayor libertad de movimiento en las retracciones y gracias a sus caracteristicas, el uso de estos materiales con el sistema bowden nos suponen un problema demasiado grande y practiamente impidiendo su trabajo correctamente.

SISTEMA EXTRUSION DIRECTA El caso de tener un sistema de extrusion directa, nos da libertad a un mayor uso de materiales sin tantas complicaciones y con una commodidad mayor para las diferentes configuraciones de cada pieza, ya que obtenemos mayor libertad para el tema de las retracciones, permitiendo que el filamento se retraiga de manera comoda y evitando atascos, dentro de las impresoras con este tipo de sistema encontramos algunas como;
  1. Creality Ender 3 S1, SE y KE
  2. Prusa MK3S
  3. Kingroon KP3S
  4. Anycubic Kobra
Gracias a este tipo de sistema, podemos utilizar diferentes tipos de materiales sin problema alguno, desde el mas comun como lo es el PLA, pasando por ABS y por supuesto, el TPU flexible; que es un material bastante complejo de manejar debido a su flexibilidad
